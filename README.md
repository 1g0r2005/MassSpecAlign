MassSpecAlign - небольшой инструмент, написанный на Python, позволяющий проводить простой статистический анализ и оценивать качество множественного выравнивания масс-спектров, в первую очередь ориентированный для использования при подготовке MSI ([mass spectrometry imaging](https://en.wikipedia.org/wiki/Mass_spectrometry_imaging)).

* Данные загружаются в формате HDF файлов.
* Предобработка для удаления артефактов
* Построение консенсусов используя ядерную оценку плотности (KDE) - библиотека [KDEpy](https://pypi.org/project/KDEpy/)
* Сопоставление консенсусов с помощью венгерского алгоритма
* Проверка изменения дисперсии для отдельных пиков через тест Левене и обобщение результата с использованием среднего гармонического p-value
---
**Версии**

* main - имеет графическую оболочку, созданную с помощью PyQt, позволяет визуализировать KDE и сопоставление пиков, а также предоставляет всю рассчитываемую информацию в удобном виде - для использования в качестве отдельного инструмента
* cli - версия для работы в консоли, минималистична и подходит, например для интеграции в собственные пайплайны.
---
**Зависимости**

Все зависимости перечислены в файлах requirements.py - каждый файл соответствует одной из версий программы.
